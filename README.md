# Codante Scripts

## Instalação

Rode o comando

```bash
bun install
```

## Adicione uma chave do Github

Crie uma chave pessoal (personal token classic) no github. Copie o `.env.copy` para `.env` e adicione-a lá.

## Rode os scripts

Complete o que existe no `index.ts` e rode com o `bun`:

```bash
bun run index.ts
```
